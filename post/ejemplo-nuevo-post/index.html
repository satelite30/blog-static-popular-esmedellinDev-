Ejemplo de Nuevo Post

Este es un ejemplo de cómo crear un nuevo post en Markdown

Publicado el 24 de Marzo de 2026 · Autor: Equipo Huellas de Barrio

Introducción al Post de Ejemplo

Este es un post de ejemplo que muestra cómo crear nuevas entradas en el blog usando archivos Markdown.

Cada archivo .md que crees en src/content/blog/ se convertirá automáticamente en una página del blog.

Cómo Funciona

El sistema de contenido de Astro lee todos los archivos Markdown de la carpeta blog y los transforma en páginas estáticas.

Cada campo en el frontmatter (la sección entre ---) define los metadatos del post.

Puedes agregar múltiples párrafos de texto secundario simplemente añadiendo más elementos al array.

Ver Documentación de Astro

Conclusión

Con este sistema, agregar nuevos posts es tan simple como crear un archivo Markdown.

No necesitas tocar código, solo editar el contenido del archivo.
